## About

Website for Suryavansham Palace — a guesthouse in Ayodhya, Uttar Pradesh, located approximately 20 minutes from the Ram Mandir. Serves pilgrims, families, and tourists visiting Ayodhya.

Live at [suryavanshampalace.in](https://suryavanshampalace.in)

## Run Locally

No build step — just open in a browser:

```bash

cd suryavansham-palace-website
# Open index.html in your browser
```
